
# Creacion de un Entorno Virtual
```bash
Crear un entorno virtual:
python -m venv .env
```

## Activar Entorno Virtual
```bash
cd .env/Scripts
activate
```

## Actualizar Entorno Virtual

```bash
python -m pip instapp --upgrade pip
```

# Instalar Psycopg2
```bash
python3 -m pip install psycopg2-binary
```
