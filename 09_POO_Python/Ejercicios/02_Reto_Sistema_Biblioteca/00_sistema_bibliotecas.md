# RETO: Sistema de Biblioteca

---

## Definición
Crear un Sistema de Bibliotecas, cada Objeto de la Biblioteca almacena libros libros, aplicar POO para resolver este ejercicio.

## Instrucciones
1. Un Libro debe tener los siguientes campos (Encapsulamiento):

    - Titulo
    - Autor
    - Genero

2. Una Biblioteca debe ser una Lista de libros, se deben crear los siguientes métodos para administrar los libros (Encapsulamiento):

    - Agregar libros
    - Buscar libros por Autor
    - Buscar libros por Genero
    - Mostrar todos los libros
    - Mostrar un libro
